## 📖 Содержание

### Практические разделы
- 💣 **Разминирование** — типы ВУ, СВУ через /try и /do, таймер
- 🪢 **Растяжка** — обезвреживание натяжных устройств
- 🚑 **Первая помощь** — все виды ПДП (пулевое, переломы, кровотечения)
- 🔫 **Штурм** — группы, виды, тактики («Крючок», «Крест», «Угол», «Диагональ»)
- 🤝 **Переговоры** — цели, обязательные действия, запреты
- 🚗 **ДТП** — полный порядок оформления (4 части + финал)

### Теоретические разделы
- 📚 **Теория** — 5 полных лекций СР ДПС
- ⛓️ **Задержание** — алгоритм и заполнение протокола
- 🎖️ **Субординация** — правила обращения и замена слов
- 📋 **Состав** — таблица личного состава (если же вам она будет нужна)
